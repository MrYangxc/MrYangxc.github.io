ubuntu学习笔记

ubuntu

Linux常用命令

1、文件列表ls（list）

ls 查看当前目录

ls /home/a 查看a目录

ls / 查看根目录

ls -l /home/a 以详细模式查看

2、目录切换cd（change directory）

pwd 显示当前工作目录

cd 切换到主目录

cd /home/a 切换到a目录

几种特殊目录：

  • ~ 代表当前用户的主目录
  • . 代表当前目录
  • .. 代表上一级目录

3、目录操作：mkdir创建目录、rmdir删除目录、cp复制、mv移动

mkdir a 创建a目录

mkdir -p a/b/c 创建多级目录

modir a 删除a目录，注意：如果目录非空，则删除失败

rm -rf a 删除a目录，和子项一并删除

cp -rf a a2 复制文件或者目录a到a2

mv a a2 将a移动到a2或者重命名为a2

4、归档tar（tape archive）

tar -cvf xxx.tar file1 file2 file3 创建档案包（c：create创建档案 v：verbose显示详情 f：file文件）

tar -xvf xxx.tar 解压缩

tar -xvf xxx.tar -C dir 解压缩到指定目录

直接cvf归档文件过大，可以进行压缩：

  • tar -zcvf xxx.tar.gz file1 file2 file3 压缩
  • tar -zxvf xxx.tar.gz 解压缩

5、软链接ln（link）（类似Windows的快捷方式）

ln -s source link s表示soft（默认为硬）将source创建软链接link

通过ls -l可以查看该文件是否为软链接

6、用户管理（所有用户操作都需要管理员权限才能执行的）

sudo useradd -m test1 sudo表示管理员权限运行，m表示在/home目录下添加用户目录

sudo passwd test1 修改用户密码

sudo userdel test1 删除用户

7、超级用户

sudo passwd root 首次使用，需要给root设置密码

su root 切换到root用户（su：switch user）

exit 退出

8、用户组

groupadd groupa 创建组groupa

useradd -m -g groupa test1 将test1添加到组groupa

usermod -g groupa test1 修改用户信息

cat /etc/group 查看用户和组

cat /etc/passwd 查看用户列表

9、文件的权限

owner 文件的属主owner

r 文件是否可读read

w 文件是否可写write

x 文件是否可以执行 excute

格式：十个字符

  • 第一个字符：d 目录：- 文件：l 软链接
  • 其余九个：权限分为三部分：自己|同组|别人 例如：rwxrw-r–，自己read，write，excute，别人read

10、修改文件访问权限chmod

chmod a+w a 对a文件所有人all添加w权限

chmod a-w a 对a文件所有人all减去w权限

​ a 所有人all

​ u 用户user自己

​ o 别人other

只有文件的属主或者root可以修改文件访问权限

省略写法：chmod +w a 默认修改自己和本组的权限

11、环境变量

export OUTDIR=/opt/ 定义环境变量

echo ${OUTDIR} 显示环境变量

printenv 查看所有环境变量

12、用户环境变量（定义在~/.profile中）

ls -la 查看所有文件，因为profile是隐藏文件不能被直接查看

在Linux中，以.开头的文件是隐藏的

13、系统环境变量（定义在/etc/profile中）

gedit /etc/profile.d/a.sh 在profile.d中创建一个shell脚本文件

export OUTDIR=/opt/ 定义环境变量

14、PATH环境变量（用于描述可执行文件的搜索路径）

export PATH=$PATH:路径 定义PATH环境变量

15、手动配置网络

​ 默认的，Ubuntu不带ifconfig命令，需要使用apt软件包管理器下载

apt install tool 安装一个软件包

apt remove tool 删除一个软件包

apt search tool 搜索

apt list | grep tool 列表

ifconfig 查看网络配置

sudo ifconfig xxx up 开启网络

sudo ifconfig xxx down 关闭网络

16、FTP服务器

service vsftpd start 启动FTp服务器

service vsftpd status 查看服务状态

service vsftpd stop 停止服务

service vsftpd restart 重启服务

17、vim文本编辑器

vim a.txt 打开a文件，如果不存在则会新建

i键 编辑模式insert mode

esc键 命令模式command mode

:wq 保存并退出

:q 退出

:q! 强制退出
